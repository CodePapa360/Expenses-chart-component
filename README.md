- View the bar chart and hover over the individual bars to see the correct amounts for each day

## Installation

- Clone this repo:

```sh
git clone https://github.com/CodePapa360/Expenses-chart-component.git
```

- Install dependencies:

```sh
npm install
```

- Build command:

```sh
npm run build
```

- Live server:

```sh
npm start
```
